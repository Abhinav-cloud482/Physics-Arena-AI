# Physics-Arena-AI

A real-time physics engine with AI-controlled behavior and collision dynamics using Pygame.


## Physics Arena AI

A real-time 2D physics simulation built with Python and Pygame, featuring collision detection, gravity, and simple AI behavior.

This project demonstrates core game development concepts such as motion physics, object interaction, and AI-driven movement in a minimal, single-file implementation.


## Features

* Real-time physics simulation (gravity, velocity, motion)
* Collision detection with realistic bounce response
* AI-controlled enemy that chases the player
* Player-controlled movement using keyboard input
* Demonstrates core game development principles


## Tech Stack

* Python 3
* Pygame


## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/physics-arena-ai.git
cd physics-arena-ai
```

### 2. Install dependencies

```bash
pip install pygame
```

### 3. Run the project

```bash
python simulation.py
```


## Controls

| Key      | Action     |
| -------- | ---------- |
| Left  | Move left  |
| Right | Move right |
| Up    | Move up    |
| Down  | Move down  |

* You control the **green ball**
* The **red ball (AI)** will chase you
* Blue balls move with physics and collide with everything


## Concepts Demonstrated

### Physics Simulation

* Gravity acceleration
* Velocity-based movement
* Energy loss on collisions

### Collision Detection

* Circle-based collision checks
* Momentum exchange between objects

### AI Behavior

* Target tracking using vector math
* Continuous pursuit mechanics

## Limitations

This is a simplified simulation and does not include :-

* Advanced physics (rotation, friction, constraints)
* Pathfinding algorithms (like A*)
* Performance optimizations (spatial partitioning)
* 3D rendering
